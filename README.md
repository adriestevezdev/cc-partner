# CC Partner

Skills para construir software con IA - desde cero hasta producción.

## Instalación

```bash
claude plugins install cc-partner@cc-partner-marketplace
```

## Skills disponibles

| Skill | Comando | Descripción |
|-------|---------|-------------|
| Empezar | `/empezar` | Inicia un proyecto desde cero con guía paso a paso |
| Brainstorm | `/brainstorm` | Diseña features y funcionalidades antes de implementar |
| Planificar | `/planificar` | Divide tareas complejas en subtareas manejables |
| Ejecutar | `/ejecutar` | Ejecuta tareas en paralelo con subagentes |

## Para quién es

- **Principiantes**: Guía paso a paso, explicaciones claras, recomendaciones de tecnologías
- **Desarrolladores**: Workflows profesionales, best practices, automatización

## Uso

```
> /empezar
```

El skill te guiará con preguntas para entender qué quieres construir y te ayudará a estructurar tu proyecto.
